Kết quả kỳ thi học sinh giỏi quốc gia năm học 2014

Lần cập nhật cuối lúc Thứ sáu, 24 Tháng 1 2014 12:01 Viết bởi Administrator Thứ sáu, 24 Tháng 1 2014 11:42

 

Trong toàn tỉnh có tổng số 22 giải quốc gia, số lượng giải của Trường THPT chuyên Nguyễn Bỉnh Khiêm đạt 18 giải. Trong đó có một giải nhất môn Toán của học sinh Võ Quang Hưng, lớp 11 chuyên toán của Trường THPT chuyên Nguyễn Bỉnh Khiêm.


Dưới đây là thống kế số lượng giải học sinh giỏi quốc gia chuyên Nguyễn Bỉnh Khiêm năm 2014 như sau:

SL Nhất Nhì Ba KK
Toán 3 1 0 1 1
Vật Lý 3 0 0 1 2
Hóa học 4 0 1 2 1
Tin học 0 0 0 0 0
Sinh học 0 0 0 0 0
Ngữ Văn 2 0 0 0 2
Lịch sử 5 0 3 0 2
Tiếng Anh 1 0 0 0 1
Tổng 18 1 4 4 10
